## 📝 Example Usage

To update personal information:
```json
{
  "firstName": "Your Name",
  "lastName": "Your Surname",
  "title": "Your Title",
  "company": "Your Company"
}
```

To add new expertise area:
```json
{
  "title": "New Expertise",
  "description": "Description of expertise",
  "icon": "🚀",
  "gradient": "from-blue-500 to-purple-600",
  "skills": ["Skill 1", "Skill 2"]
}
```
